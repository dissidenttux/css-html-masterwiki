CSS/HTML MasterWiki

Learn CSS/HTML in 5-20 hours

Inspired by HackerHero's Course

The point of this page is to provide an easy reference for HTML elements and CSS properties, and learn HTML and CSS within 5-20 hours as I build this site.

Here's a list of the "Super Important" HTML elements that you could build 99% of websites in the world with:

  • Headings <h1><h2><h3>
  • Paragraphs <p>
  • Anchors ("Links") <a>
  • Images <img>
  • Lists... <ul>(Unordered), <ol>(Ordered), <li>(List Items)
  • Divisions and Spans <div><span>
  • Tables <table><tr><td>
  • Forms <form>
  • Input <input>
  • Form Fields <textarea>
  • Drop-Down Lists <select><option>
